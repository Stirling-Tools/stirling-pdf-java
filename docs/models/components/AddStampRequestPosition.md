# AddStampRequestPosition

Position for stamp placement based on a 1-9 grid (1: bottom-left, 2: bottom-center, 3: bottom-right, 4: middle-left, 5: middle-center, 6: middle-right, 7: top-left, 8: top-center, 9: top-right)


## Values

| Name    | Value   |
| ------- | ------- |
| `ONE`   | 1       |
| `TWO`   | 2       |
| `THREE` | 3       |
| `FOUR`  | 4       |
| `FIVE`  | 5       |
| `SIX`   | 6       |
| `SEVEN` | 7       |
| `EIGHT` | 8       |
| `NINE`  | 9       |
