# MergeMultiplePagesRequestPagesPerSheet

The number of pages to fit onto a single sheet in the output PDF.


## Values

| Name      | Value     |
| --------- | --------- |
| `TWO`     | 2         |
| `THREE`   | 3         |
| `FOUR`    | 4         |
| `NINE`    | 9         |
| `SIXTEEN` | 16        |
