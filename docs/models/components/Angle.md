# Angle

The angle by which to rotate the PDF file. This should be a multiple of 90.


## Values

| Name                      | Value                     |
| ------------------------- | ------------------------- |
| `ZERO`                    | 0                         |
| `NINETY`                  | 90                        |
| `ONE_HUNDRED_AND_EIGHTY`  | 180                       |
| `TWO_HUNDRED_AND_SEVENTY` | 270                       |
