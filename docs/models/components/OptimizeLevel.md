# OptimizeLevel

The level of optimization to apply to the PDF file. Higher values indicate greater compression but may reduce quality.


## Values

| Name    | Value   |
| ------- | ------- |
| `ONE`   | 1       |
| `TWO`   | 2       |
| `THREE` | 3       |
| `FOUR`  | 4       |
| `FIVE`  | 5       |
| `SIX`   | 6       |
| `SEVEN` | 7       |
| `EIGHT` | 8       |
| `NINE`  | 9       |
